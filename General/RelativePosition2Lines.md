# Relative Positions of 2 lines


## OA: (xA - xO) / (yA - yO)
## OB: (xB - xO) / (yB - yO)

```
O------A---------------B
```
## OA and OB are collinear: (xA - xO) * (yB - yO) - (xB - xO) * (yA - yO) = 0

```
A------O
      /
    /
  /
B
```
## OA is clock-wise to OB: (xA - xO) * (yB - yO) - (xB - xO) * (yA - yO) > 0


```
B
  \
    \
      \
        \
A--------O
```
## OA is counter clock-wise to OB: (xA - xO) * (yB - yO) - (xB - xO) * (yA - yO) < 0
